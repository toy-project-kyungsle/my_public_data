# 3. 노트

전략을 어떻게 하면 잘 짤 수 있을까?
일단 오늘은 해야 하는 것들이 많아서 이 것들을 깔끔하게 처리해서 끝내는 방법을 고려하자.
매몰되지 않고 어떻게 하면 기술적으로 빠르고 깔끔하게 끝낼 수 있는지 고민하자.
==시간을 만들자 여유롭게 하자.==

조급한 마음을 버리고 여유롭게 진행하자.
조급하면 아무 것도 못한다.
하루에 한시간 낼 수 있을 만큼의 여유를 가지자.


- 어드민 토큰 관련해서 왜 안 되었는지 로직
	- 어떻게 하면 깔끔하게 마무리 할 수 있을까? 로직을 찬찬히 살펴보자. 
	- 내 생각에 accesstoken이 401이 뜨면 refresh token으로 재발급을 시키고, refresh token으로도 재발급이 안 되면 로컬스토리지를 비워주는 간단한 로직이다. 비워주면 알아서 페이지 어드민 로그인 페이지로 넘긴다.
	- 이거 AI가 코드를 적어줬는데 이걸 잘 리뷰하면 된다

- 동영상 배너 끝내기
	- 이건.. 아이디어 잘 내어가지고 끝내야겠다 너무 밀리는 것 같다
	- 사실 우선순위가 높은 것 같지는 않지만!
