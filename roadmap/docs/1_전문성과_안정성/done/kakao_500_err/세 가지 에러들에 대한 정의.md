**1.** `**"the account is inactive."**` **— 계정 비활성**
공식 에러코드 `**-103**` (HTTP 400)에 해당해. 카카오 로그인 에러 코드 문서에 따르면 **"휴면 상태, 또는 존재하지 않는 카카오계정으로 요청한 경우"** 발생한다고 되어 있어. 즉, 장기간 미사용으로 카카오계정이 휴면 처리됐거나 탈퇴/삭제된 계정으로 API를 호출했을 때 나오는 에러야.
  
**2.** `**"given account is not connected to any talk user."**` **— 카카오톡 미연결**
공식 에러코드 `**-501**` (HTTP 400). 공통 에러 코드에 **"카카오톡 미가입 또는 유예 사용자가 카카오톡 또는 톡캘린더 API를 호출한 경우"**라고 정의되어 있어. 카카오계정은 있지만 카카오톡에 가입하지 않았거나, 카카오톡을 탈퇴/유예한 상태에서 카카오톡 관련 API(메시지 보내기, 채널 조회 등)를 호출하면 발생해. 실제로 카카오톡 채널 REST API 문서에서도 이 에러 응답 예시를 보여주고 있어.
  
**3.** `**"NotRegisteredUserException"**` **— 미등록 사용자**
공식 에러코드 `**-101**` (HTTP 400). 카카오 로그인 에러 코드에 **"해당 앱에 카카오계정 연결이 완료되지 않은 사용자가 호출한 경우"**라고 되어 있어. Android SDK 레퍼런스에서도 `NotRegisteredUser(-101)`로 명시되어 있고. 카카오 로그인을 통해 앱에 연결(가입)하지 않은 사용자의 토큰이나 ID로 API를 호출했을 때 발생하는 에러야. 해결하려면 카카오계정 연결을 먼저 완료해야 해.
